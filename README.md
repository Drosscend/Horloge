# Juste une horloge

## Installation

```bash
git clone https://github.com/Drosscend/Horloge
cd Horloge
```

Lancer le fichier `index.html` dans votre navigateur.

## Utilisation

Au lancement, l'horloge affiche l'heure actuelle.

## Technologies utilisées

- Tailwind CSS (https://tailwindcss.com/)
